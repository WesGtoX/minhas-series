## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/download/)
- [Yarn](https://classic.yarnpkg.com/en/docs/install/)

### Install and Run

1. Clone the repository:
```bash
git clone https://github.com/WesGtoX/minhas-series.git
```
2. Install the dependencies:
```bash
yarn install
```
3. Run:
```bash
yarn start
```
